# Épicos — FinTrack

## EPIC-01 — Gestão de Categorias

**Objetivo:** permitir ao usuário criar e visualizar categorias para
classificar transações financeiras (ex.: Alimentação, Transporte, Salário).

**Features incluídas:** F-01

**Status:** Done

---

## EPIC-02 — Gestão de Transações

**Objetivo:** permitir ao usuário registrar transações financeiras
(entradas e saídas) vinculadas a categorias, e visualizar o histórico
de movimentações.

**Features incluídas:** F-02

**Status:** Done

---

## EPIC-03 — Relatórios Financeiros

**Objetivo:** oferecer visões consolidadas das finanças do usuário —
saldo atual (entradas − saídas) e distribuição de valores por categoria.

**Features incluídas:** F-03, F-04

**Status:** Done
