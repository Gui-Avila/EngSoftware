# FinTrack — Gerenciador Financeiro Pessoal

Trabalho A2 de Engenharia de Software (PUC-Rio). Aplicação web em Flask com Clean Architecture, SOLID e testes unitários com pytest.

Registra transações financeiras (entradas e saídas), organiza por categoria, calcula saldo e totaliza por categoria.

## Como rodar

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python run.py
```

## Como testar

```bash
pytest
pytest --cov
```

## Lint

```bash
ruff check .
```
